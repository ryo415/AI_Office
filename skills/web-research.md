# Web Research Skill

## Purpose

Research current or external information.

## Input

- Research question
- Required freshness
- Evaluation criteria
- Constraints

## Rules

- Use official sources when possible.
- Use recent sources for fast-changing topics.
- Cite sources when used.
- Separate confirmed facts from assumptions.
- Capture enough source metadata for future Notion or GitHub reuse.

## Output

```markdown
## Research Question

## Summary

## Findings

## Comparison

## Recommendation

## Sources

## Uncertainty

## Next Actions
```
